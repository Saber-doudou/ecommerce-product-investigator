## 五、利润测算（跨境版专用，国内版省略）

### 商品A 利润链

| 项目 | 数值 | 说明 |
|------|------|------|
| 采购价 | ¥XXX = $XXX | 1688/买手价 |
| 头程运费 | $X.X | 按实际物流商报价 |
| FBA/平台费 | $X.X | 按平台费率计算 |
| 平台佣金 | 售价 × X% = $X.X | 基于售价扣除 |
| 税金（VAT/销售税） | 售价 × X% = $X.X | 基于售价扣除 |
| **单位硬性成本** | **$XX.X** | 采购+头程+FBA+佣金+税金 |
| **保本售价** | **$XX.X** | 成本 ÷ (1 - 佣金率 - 税率) |
| **实际售价** | **$XX.X** | 市场调研得出 |
| **单件利润** | **$X.X** | 售价 - 成本 - 佣金 - 税金 |
| **利润率** | **XX.X%** | 利润 ÷ 售价 |

> ⚠️ 利润计算公式：保本售价 = 单位硬性成本 ÷ (1 - 平台佣金率 - 税率)
> 佣金和税金均基于售价扣除，不是基于成本。

| 场景 | 售价变动 | 利润率 | 说明 |
|------|---------|--------|------|
| 乐观（竞品少） | +10% | XX% | 可阶段性提价 |
| 基准 | $XX.X | XX% | 当前市场价 |
| 悲观（竞品涌入） | -10% | XX% | 需降本或退出 |
