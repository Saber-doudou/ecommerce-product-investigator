# 平台差异速查指南（跨境专用）

> ⚠️ **本文件仅适用于跨境模式**（Amazon/Shopee/TikTok Shop/Temu）。国内电商平台的参数提取规则请参考 `references/domestic-guide.md`。

不同电商平台的规则差异很大，用同一套分析模板套不同平台会产生错误结论。本文档列出关键差异和对应策略。

---

## Amazon vs Shopee vs TikTok Shop vs Temu 关键差异

| 维度 | Amazon | Shopee | TikTok Shop | Temu |
|:---|:---|:---|:---|:---|
| 佣金结构 | 15% (多数品类)，另有 variable closing fee | 5-6% + 2% payment fee | 5% + 1% payment fee（新卖家优惠期更低） | 0-5%（目前补贴期） |
| FBA/物流 | FBA 费用按尺寸分段，仓储费月结 | 官方物流 SLS，费用较低 | 平台物流或自发货 | 全托管模式，卖家不控物流 |
| Review 机制 | 评分权重受验证购买影响，Vine 计划 | 评分简单平均，刷评管控较松 | 评分与视频内容强关联 | 评分体系尚不成熟 |
| BSR 算法 | 综合销量+近期增速，每小时更新 | 基于近30天销量 | 基于视频互动+转化 | 基于全平台销量排序 |
| 广告成本 | CPC 较高 ($0.5-3+)，竞争激烈 | CPC 较低，但流量也少 | 达人佣金 + 广告双成本 | 暂无卖家自助广告 |
| 退货率 | 多数品类 5-15%，服装可达 20%+ | 相对较低 3-8% | 服装 10-20%，其他 5-10% | 平台承担退货 |
| 入场门槛 | 需要品牌备案/UPC，合规要求高 | 入场容易，但同质化严重 | 需要达人资源或短视频能力 | 全托管，需供货资质 |
| 促销节奏 | Prime Day、黑五为核心节点 | 双日大促 (9.9/11.11/12.12) | 超级品牌日 + 大促联动 | 全年低价策略 |

## Amazon 各站点差异

| 维度 | US | DE | JP | UK |
|:---|:---|:---|:---|:---|
| 默认佣金 | 15% | 15% | 15% | 15% |
| VAT/税 | 无 (销售税各州不同) | 19% VAT | 10% 消费税 | 20% VAT |
| FBA费用 | 中等 | 较高 | 中等 | 较高 |
| Review 门槛 | 高 | 高 | 中 | 高 |
| 竞争强度 | 最高 | 中高 | 中 | 中高 |
| 日均单量潜力 | 最大 | 中等 | 中等 | 中等 |

⚠️ **做利润测算时，VAT 是很多卖家忽略的成本**——德国 19%、英国 20% 的 VAT 从售价扣除，直接影响利润。

## 买手 API 策略（跨境模式）

`scripts/maishou_price.py` 主要用于获取国内采购端参考价，适用于：

- **Amazon 采购成本估算**：用 1688/买手价作为采购基准
- **Shopee 采购成本估算**：同上，但需额外考虑 SLS 物流费
- **多市场对比**：同一产品在不同市场的利润差异

### API 使用注意

1. 买手价格反映的是国内批发/零售端，不是工厂出厂价——利润测算中留 10-20% 的议价空间
2. API 返回的价格可能包含平台补贴，实际采购价可能更高
3. 关键词搜索建议用中文，匹配效果更好
4. 跨境卖家常犯的错：用买手零售价当采购成本，导致利润测算虚高

## 平台适配检查清单

在输出报告前，确认以下项目是否按目标平台调整：

- [ ] 佣金率使用了正确平台的费率
- [ ] 税费计算用了目标市场的税率（VAT/Sales tax）
- [ ] FBA/SLS 物流费用了对应平台的计费标准
- [ ] BSR 排名参考了对应站点的数据
- [ ] 促销节点匹配了目标市场的节奏
- [ ] 退货率估算参考了对应平台的行业数据
